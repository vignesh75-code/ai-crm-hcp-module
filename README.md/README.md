# AI CRM HCP Interaction Module

## Overview
The AI CRM HCP Interaction Module is an AI-powered assistant designed to help pharmaceutical sales representatives interact with Healthcare Professionals (HCPs).  
The system uses an AI agent to respond to queries, retrieve product information, and assist with CRM interaction workflows.

## Features
- AI-powered query handling
- LangGraph agent architecture
- Tool-based modular system
- FastAPI backend
- Simple web frontend interface

## Tech Stack

Backend
- Python
- FastAPI
- LangGraph
- Groq LLM API

Frontend
- HTML
- JavaScript
